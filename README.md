# 📺 Teleprompter Pro

Una aplicación web de teleprompter profesional, similar a "Teleprompter for Video" de App Store.

## ✨ Características

- **Editor de Scripts**: Escribe o pega tu guion con contador de caracteres y tiempo estimado
- **Modo Teleprompter**: Scroll suave del script mientras grabas
- **Controles Completos**:
  - Ajuste de velocidad de scroll (1-10)
  - Tamaño de fuente ajustable (24-120px)
  - Márgenes personalizables
  - Modo espejo para rigs profesionales
  - Modo oscuro/claro
- **Cuenta Regresiva**: Countdown de 3 segundos antes de iniciar
- **Atajos de Teclado**:
  - `Espacio`: Iniciar/Pausar
  - `↑/↓`: Ajustar velocidad
  - `Escape`: Salir
- **Persistencia**: Tu script y configuración se guardan automáticamente

## 🚀 Uso

1. Abre `index.html` en tu navegador
2. Escribe o pega tu script en el editor
3. Ajusta la configuración (velocidad, tamaño, etc.)
4. Haz clic en "🎬 Iniciar Teleprompter"
5. Presiona "▶️ Iniciar" y lee tu script mientras se desplaza
